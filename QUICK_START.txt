HOW TO USE THE APPLICATION:
====================================================================

STEP 1: INITIALIZE DATABASE (One time only)
   python backend/database/setup_db.py

STEP 2: START APPLICATION
   python backend/app.py

STEP 3: OPEN BROWSER
   http://localhost:5000

STEP 4: REGISTER ACCOUNT
   - Click "Create one" on login page
   - Enter username, email, password
   - Click "Create Account"

STEP 5: LOGIN
   - Enter username and password
   - Click "Sign In"

STEP 6: ADD YOUR BUDGET/SALARY
   - Click "Add Transaction"
   - Select "Income"
   - Enter amount (e.g., 5000)
   - Description: "Monthly salary"
   - Category: "Salary"
   - Click "Add Transaction"

STEP 7: ADD EXPENSES
   - Click "Add Transaction"
   - Select "Expense"
   - Enter amount (e.g., 15.50)
   - Description: "Starbucks coffee"
   - AI will suggest "Food & Dining"
   - Click "Add Transaction"

STEP 8: VIEW DASHBOARD
   - See your balance (Income - Expenses)
   - Green if positive (within budget)
   - Red if negative (over budget)
   - View charts and recent transactions

SUPPORT & TROUBLESHOOTING:
====================================================================

Common Issues:

"Module not found"
→ Run: pip install -r requirements.txt

"Database locked"
→ Delete budget_tracker.db and re-run setup_db.py

"Port already in use"
→ Change port in app.py (line 275)

"AI not working"
→ Wait for model to train (first run takes ~30 seconds)

"Can't login"
→ Check username/password
→ Clear browser cache
→ Reinitialize database
